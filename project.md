# Tryphasing

Tryphasing est un système local de pack opening TCG pour Twitch.

Les viewers peuvent acheter un pack via des points de chaîne.

Chaque pack contient 1 seule carte.

Une animation s'affiche en live sur le stream.

Le système doit :

- Gérer cartes et sets
- Gérer collection par viewer
- Ne pas compter les doubles dans les statistiques
- Être extensible facilement (ajout cartes/sets)
- Avoir une interface admin simple
- Être développé avec des données placeholder initiales
